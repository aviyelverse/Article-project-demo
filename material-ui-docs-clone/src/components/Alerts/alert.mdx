# Alert Warnings

Provide contextual feedback messages for typical user actions with the handful of available and flexible alert messages.

## Properties

<Props of={AlertBehaviour} />

## Basic usage

<Playground>
  <AlertBehaviour>Hello there ! I am a `Primary` Alert</AlertBehaviour>
</Playground>

## Examples

Alerts are available for any length of text, as well as an optional dismiss button. For proper styling, use one of the eight required contextual behaviour (e.g., ""primary" | "secondary" | "success" | "warning" | "info" | "light" | "dark" |).

<Playground>
  <AlertBehaviour behaviour="primary">
    Hello there ! I am an Alert
  </AlertBehaviour>
  <br />
  <AlertBehaviour behaviour="secondary">
    Hello there ! I am an Alert
  </AlertBehaviour>
  <br />
  <AlertBehaviour behaviour="success">

  <br />
  <AlertBehaviour behaviour="warning">

  <AlertBehaviour behaviour="info">

  </AlertBehaviour>
  <br />
  <AlertBehaviour behaviour="light">
    Hello there ! I am a "light" Alert
  </AlertBehaviour>
  <br />
  <AlertBehaviour behaviour="dark">
    Hello there ! I am a "dark" Alert
  </AlertBehaviour>
</Playground>
